Now let's create a simple UIView subclass that displays a colored rectangle. We'll implement drawRect:, as an exercise, even though the styling could be achieved by other means.

Inside drawRect:, we first create a UIBezierPath object that describes a rounded rectangle matching our bounding rectangle. Next the line width of the path is set. After setting the fillColor and strokeColor, the path is first filled and then stroked.

In this example, it's important to remember the Painter's Model. If the path were stroked first and then filled, the stroke color would not show because the fill color was painted right over it. Alternatively, we first fill the path and then stroke the path, painting the stroke color on top of the fill color.
